Синтаксис входного языка состоит из набора определенных команд либо с фиксировнным
числом параметров, либо команд инициализирующих свой блок. Сейчас выходная программа
имеет синтаксис языка C.

Список команд:

	SENDSIG N (N - 0 или 1) - Посылает сигнал на Arduino
	
	MOVE N (N - UP, DOWN, или OUT) - Совершает движиние над клеткой. UP - из нижней
точки в верхнюю, DOWN - из верхней в нижнюю, OUT - из верхней точки над клеткой в
точку над коробкой с фигурами.  Над какой клеткой совершать движение определяется
исходя из того, какие сигналы в данный момент пришли с Arduino к манипулятору.

	TAKE  - Сжать захват
	
	PUT   - Разжать захват
	
	HOME  - Встать в положение, в котором робот будет ожидать хода
	
	CLOCK - Нажать на часы
	
	CASECMD M N (M, N - значения двух старших разрядов, переданных с Arduino (0 или 1)) -
Условие, в случае выполнения которого выполняется блок последующих команд до ENDCASE
	
	CASEFULLCMD A B C D E F G H (Значения всех восьми разрядов) - Делает то же самое,
что и CASECMD, но проверяет все восемь разрядов. Нужен для обработки специальных команд,
приходящих с Arduino (Например, нажатие на часы).
	
	END. - Завершение кода программы
	
	LOOP - Начало блока бесконечного цикла. Предполагается, что этот блок идет самым
последним в программе, поэтому он закрывается вместе с тегом END.


Примерный вид программы для робота:

SENDSIG 1
LOOP
	CASECMD 1 0	 SENDSIG 0  MOVE DOWN  TAKE  MOVE UP  				 SENDSIG 1  ENDCASE
	CASECMD 0 1	 SENDSIG 0  MOVE DOWN  PUT   MOVE UP  				 SENDSIG 1  ENDCASE
	CASECMD 1 1	 SENDSIG 0  MOVE DOWN  TAKE  MOVE UP  MOVE OUT  PUT  SENDSIG 1  ENDCASE
	
	CASEFULLCMD  0 0 0 0 0 0 0 1  SENDSIG 0  HOME   SENDSIG 1  ENDCASE
	CASEFULLCMD  0 0 0 0 0 0 1 0  SENDSIG 0  CLOCK  SENDSIG 1  ENDCASE
END.
